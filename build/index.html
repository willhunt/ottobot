# Hardware List

A summary of the main hardware:

  • Dagu Wild Thumper 4wd with 75:1 gear motors [Product Page, Useful Info]
  • Raspberry Pi 3 B+ [Product Page]
  • Arduino M0 Pro [Product Page, Guide]
  • Adafruit BNO055 9DOF IMU [Guide, Datasheet]
  • Cytron 10Amp 5V-30V DC Motor Driver (2 Channels) [Product Page]
  • Zippy 5000mAh LiPo battery [Product Page, User Manual]
  • DFRobot DC-DC Power Module [Product Page, Datasheet]
