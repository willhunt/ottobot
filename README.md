# Ottobot
Mobile robot platform using ROS

## Hardware
A summary of the main hardware:
* Dagu Wild Thumper 4wd with 75:1 gear motors
* Raspberry Pi 3 B+
* Arduino M0 Pro
* Adafruit BNO055 9DOF IMU
* Cytron 10Amp 5V-30V DC Motor Driver (2 Channels)
