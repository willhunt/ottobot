# Specific use with Arduino M0 Pro

There are some specific notes for using the sensor with the Arduino M0 Pro

  1. Use I2C address 0x29
    • e.g: Adafruit_BNO055 bno = Adafruit_BNO055(55, 0x29);
  2. Tie ADR pin to high
